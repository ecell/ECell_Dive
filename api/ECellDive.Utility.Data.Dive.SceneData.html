Struct SceneData

Namespace
ECellDive.Utility.Data.Dive

A struct to keep track about what is hapenning in the dive. Which divers are in (or out) of the scene and the modules that are loaded.

The network serialization is partial, it does not serialize the loaded modules.

Remarks

This follows the INetworkSerializable interface from Unity Netcode for GameObject. This is necessary because this struct uses a List, which is not supported by the default serialization methods of Unity Netcode.
