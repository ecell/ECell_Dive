RefreshLineEndPosition()

Updates the end position of the LineRenderer.

  • If endPositionScope is set to local, 
    end.localPosition
    is used.
  • If endPositionScope is set to world, 
    end.position
    is used.
  • If endPositionScope is set to delta, 
    end.position - start.position
    is used.
  • If endPositionScope is set to _override, 
    overrideStart
    is used.
  • If endPositionScope is set to startLocal, 
    start.InverseTransformPoint(end.position)
    is used.
