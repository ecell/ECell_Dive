Class VolumetricSelectorManager

Namespace
ECellDive.PlayerComponents

The logic to control a collider that can be used to perform volumetric selection of objects in the scene. It can be moved forward or backward and scaled up or down. The objects that collide with it are sent to the GroupsMakingManager
