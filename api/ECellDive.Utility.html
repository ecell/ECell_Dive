AlwaysLookAt

A simple component to always call LookAt() on update.

AnchoredContinousMoveHelper

This class is used to help the user to understand the AnchoredContinousMove component. It is used to display the deadzone and the magnitude of the resulting movement vector.

AnimationLoopWrapper

A utility class to play an animation loop.

ArrayManipulation

A utility class to manipulate arrays. The purpose is to be able to fragment an array to a 2 dimensional array or a list of arrays and to assemble them back. This is useful to send data over the multiplayer network.

ColorFlash

A utility class to flash a color on an array of renderers. The flash is a sine wave defined by a period and a number of periods. The flash is done by lerping between the start colors of the renderers and the target colors.

FaceCamera

A utility component to make the gameobject it is attached to face the camera or pop up in front of the camera.

GameObjectConstructor

Spawns a prefab at a given distance in front of the camera.

LinePositionHandler

Utility class to set the start and end positions of a LineRenderer. It is particularly usefull to update the connections between an info display and its module, node, edge, etc...

LogSystem

Class handling the recording of messages to display in the log.

PerLayerCulling

A utility class to set the culling distance of each layer. This is a performance optimization.

PinSystem

A component with the logic to manipulate the parent of the gameobject it is attached to.

Positioning

Utility class to solve frequent GOs positioning problems in the scene.

ToFind

A utility class that provides a function to find a component in itself, its children gameobject and its parent (if it exists).

TriggerBroadcaster

A component allowing external scripts to subscribe to "major" Unity events (e.g. OnTriggerEnter) that are invoked on by this gameobject.
