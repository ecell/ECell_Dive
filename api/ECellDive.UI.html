Table of Contents

Namespace ECellDive.UI

Classes

ButtonColorPicker

Associates a color picker to a button. This way clicking on the button will open the color picker.

ConnectionAnchorPosition

Computes the position of an anchor in a 2D UI panel that is the closest to its master object. A connection line will be drawn between the anchor and the master object.

ContextualHelpManager

Class to link the input mode of the controllers to the content of the information tags. When the input mode changes, the information tags are updated accordingly. It also provides a method to hide all information tags at once.

FbaParametersManager

Manages the GUI elements that are part of the option panel of the HttpServerFbaAnalysisModule.

GUIManager

Manages the initialization of the GUI and the logic to open/close the Main Menu.

GroupByAttributsManager

Manages the GUI attached to a GroupByModule that the user interacts with in order to request processing groups.

GroupUIManager

Manages the GUI element exposing a Group data (name, color & activation) in the scene to the user.

GroupsMakingUIManager

The high level manager of the UI used to validate or cancel the creation of new groups.

GroupsMenu

Manages the GUI that displays every group either produced by a "GroupByModule" or made out of selected objects by the user.

InfoDisplayManager

A class to manage an information 2D UI panel that is attached to a gameobject by a connection line. The info tag can be displayed/hidden, can look at a target, and can change its text.

InfoTagManager

A specialized InfoDisplayManager that manages the display of the input controls of the player. Typically handles what it should display when the player is in a certain control mode (i.e. Interaction, Movement or Groups).

LogManager

Class handling the UI gameobject used to visualize the log in app at runtime.

MainMenuManager

The class to manage the main menu UI.

ModulesMenuManager

The class to manage the module menu UI.

MultiplayerMenuManager

The class to manage the multiplayer menu UI. It is used both to join and host a multiplayer session.

OptimizedVertScrollList

A quick implementation of a vertical scroll list that can be used to display long lists of items without causing performance issues.

SemanticGroupUIManager

Manages the GUI container (drop down) storing every GUI element representing groups.

SimpleDropDown

Manages a GUI element behaving like a drop down.

SliderValueControlManager

Manages the synchronisation within a GUI element that links a slider, with an input field and two buttons (+ and -)

SurgeAndShrinkInfoTag

A utility class to trigger a surge and shrink animation on the game object this script is attached to. Requires a reference to a text mesh to explain the surge and shrink to the player.

UIHover

Implements Haptic and Sound feedback when hovering on interactible UI elements.

VirtualKeyboardLinker

Links a Text Mesh Pro Input field to the virtual keyboard of the scene.

VirtualKeyboardManager

Simple virtual keyboard manager for QUERTY layout, with symbols, numbers and capital letters.
