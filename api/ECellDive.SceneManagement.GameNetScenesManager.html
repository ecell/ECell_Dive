modulePrefabs

A list of prefabs that may be instantiated by the user at runtime. Every prefab MUST have a GameNetModule and Unity.Netcode.NetworkObject component. We advise to use this list for root data modules (e.g. CyJsonModule) only. "Submodules" that may be instantiated as a result of a module's data generation should be referenced in the script detailing the said data generation process.
