Class DummyGameNetModule

Namespace
ECellDive.Modules

A dummy game net module. This only serves the purpose of demonstrating how to create a game net module and which methods must be implemented. It does not do anything. It is only attached to the game object called "BaseGameNetModule" for demonstration. DO NOT REUSE THIS CLASS.

RequestSourceDataGenerationServerRpc(ulong)

Requests the server to generate the data associated to the module. This MUST be implemented by any divable module in order for the data generation to be broadcasted to the other clients.

public override void RequestSourceDataGenerationServerRpc(ulong _expeditorClientID)

Parameters

_expeditorClientID ulong

The ID of the client that is requesting the generation.

Remarks

RPCs cannot be marked as abstract so we can only define the method as virtual. Unfortunately, this means that we cannot force the implementation of this method and that developpers will have to remember to implement it in their modules if it is divable.
