Class NetworkSpawnComponentActivation

Namespace
ECellDive.Utility.Multiplayer

Utility class to add to player gameobjects to activate/deactivate components depending on whether the player is local or not. Any player gameobject that is not local (i.e. it is the gamobject representing a remote player) will require slightly different components to be active than the local player.
