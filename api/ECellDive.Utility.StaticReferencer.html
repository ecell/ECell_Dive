Class StaticReferencer

Singleton class referencing elements (GameObjects, Components,...) of the player that are needed by gameobjects outside it. Typically, an instanced module needs access to the controller's interactors to correctly compute the movements when grabed. This class therefore exposes all the interactors to the ouside.
