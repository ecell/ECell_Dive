Remote Importer

Requires:

  • A connection to Kosmogora.
  • The API Checker has validated the Kosmogora server.

This is the module that is used to import the data stored on Kosmogora.

Panel Server Parameters

  1. Button Server. Will open the Server choice panel to chose among the Kosmogora-like servers which API allows to store and load data modules in ECellDive.
  2. Button Query Available Models. Will try to contact Kosmogora at the address and port given to request the list of models currently available. This is fast. If nothing happens check the Log Menu to see if an error was not raised. If the query is successful, the module will flash green; the list of available files will be displayed in panel Models List. If the query fails, the module will flash red. In that case, you can check the error message in the log.
